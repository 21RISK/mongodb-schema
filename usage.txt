Usage: mongodb-schema

Infer the probabilistic schema for a MongoDB collection.

Usage:
  mongodb-schema <uri> <ns> [--format=<json|yaml|table> --sample=<n>]

Options:
  <uri>             A MongoDB connection string.
  <ns>              The namespace of the collection to sample.
  --sample=<n>      The number of documents to sample [Default: `100`].
  --format=<name>   The output format: json, yaml, or table [Default: `json`].
  --debug           Enable debug messages.
  -h --help         Show this screen.
  --version         Show version.
